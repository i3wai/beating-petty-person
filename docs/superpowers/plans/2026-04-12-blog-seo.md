## Phase 1: Technical SEO Foundation

### Task 1.1 — Extend `PostMeta` in `src/lib/content.ts`

Add to PostMeta interface:
```typescript
cluster: "A" | "B" | "C";
updatedDate?: string;
```

Add helper functions:
```typescript
export function getPostsByCluster(locale: string, cluster: "A" | "B" | "C"): PostMeta[]
export function getRelatedPosts(locale: string, slug: string, limit?: number): PostMeta[]
```

`getRelatedPosts` returns posts with the same cluster, excluding current slug, sorted by date desc, limit 3.

### Task 1.2 — Create `src/lib/json-ld.ts`

Utility functions that return JSON-LD objects:

```typescript
export function organizationJsonLd(): object    // Organization schema
export function websiteJsonLd(): object          // WebSite schema with SearchAction
export function articleJsonLd(post: PostMeta, locale: string): object  // Article with author, publisher, dates
export function faqPageJsonLd(faqs: {question: string; answer: string}[]): object  // FAQPage
export function breadcrumbJsonLd(items: {name: string; url: string}[]): object  // BreadcrumbList
export function howToJsonLd(steps: {name: string; text: string}[]): object  // HowTo (P1)
```

All schemas use `https://beatpetty.com` as base URL. Organization: `BeatPetty`, logo from `/og-image.png`.

### Task 1.3 — Add Organization + WebSite JSON-LD to `src/app/[locale]/layout.tsx`

Insert `<script type="application/ld+json">` in the body with Organization + WebSite schemas. Use the helpers from `json-ld.ts`.

### Task 1.4 — Rewrite `src/app/[locale]/blog/[slug]/page.tsx` metadata

In `generateMetadata`:
- Set `alternates.canonical` to `https://beatpetty.com/${locale}/blog/${slug}`
- Set `alternates.languages` for en, zh-TW, zh-Hans hreflang
- Set `openGraph.type: "article"`, `publishedTime`, `modifiedTime`
- Add Article + FAQPage + BreadcrumbList JSON-LD via `<script>` tags in the page body

### Task 1.5 — Add `noindex` to `/result` page

In `src/app/[locale]/result/page.tsx`, add to metadata:
```typescript
robots: { index: false, follow: true }
```

### Verification
- `npm run build` passes
- View page source: Organization JSON-LD present in root layout
- Blog post: canonical, hreflang, Article/FAQPage/BreadcrumbList JSON-LD present
- /result page: `<meta name="robots" content="noindex, follow">`

## Phase 5: Dynamic Sitemap + Final Verification

### Task 5.1 — Replace `public/sitemap.xml` with `src/app/sitemap.ts`

Dynamic sitemap generation:
```typescript
import { MetadataRoute } from 'next';
import { getAllBlogSlugs, getAllPosts } from '@/lib/content';

export default async function sitemap(): Promise<MetadataRoute.Sitemap> {
  const locales = ['en', 'zh-TW', 'zh-Hans'];
  const entries = [];

  // Static pages
  for (const locale of locales) {
    entries.push({ url: `https://beatpetty.com/${locale}`, lastModified: new Date() });
    entries.push({ url: `https://beatpetty.com/${locale}/blog`, lastModified: new Date() });
    entries.push({ url: `https://beatpetty.com/${locale}/ritual`, lastModified: new Date() });
  }

  // Blog posts
  for (const locale of locales) {
    const posts = getAllPosts(locale);
    for (const post of posts) {
      entries.push({
        url: `https://beatpetty.com/${locale}/blog/${post.slug}`,
        lastModified: new Date(post.updatedDate || post.date),
      });
    }
  }

  return entries;
}
```

Delete `public/sitemap.xml` after verifying dynamic version works.
